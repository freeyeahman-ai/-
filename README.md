## 缘起

最近沉迷周易，无奈手伤不便操持筮竹，遂用AI辅助开发此模拟器。选用**高岛吞象**先生所创的**略筮法**（最适合初学者的入门法门），历经四十余次迭代打磨，终于达到可用状态，现开源分享。

&gt; "初筮告，再三渎，渎则不告。"  
&gt; ——《周易·蒙卦》

## 特点

- 🎋 **滑动分堆**：模拟真实的四十九根筮竹分堆过程，支持鼠标/触摸滑动
- 🕐 **真太阳时**：自动计算真太阳时与十二地支时辰（需定位权限）
- 📚 **完整卦库**：内置六十四卦详细解析，引用《白话高岛易断》
- ⚡ **动爻分析**：自动计算本卦、变卦，突出显示动爻（重要性51%）
- 🔒 **隐私至上**：纯前端实现，天机只在本地流转，不留痕、不存云端
- 📱 **响应式设计**：支持手机、平板、桌面端

## 使用方法

1. 下载 `index.html`，用浏览器直接打开（无需服务器）
2. 选择性别（男女分堆顺序不同）
3. 默念所占之事，填写于输入框
4. 滑动分堆，至诚无息时确认
5. 重复三次（上卦→下卦→动爻）
6. 查看卦象解析与动爻启示

## 技术说明

- **单文件应用**：所有代码（HTML/CSS/JS）集成在一个文件中，可直接离线使用
- **无依赖**：不依赖任何第三方库，纯原生 JavaScript 实现
- **真太阳时算法**：基于经度差与均时差（Equation of Time）计算
